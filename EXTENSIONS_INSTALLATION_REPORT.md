### Extensions Not Showing
```
Issue: Can't see installed extensions in Ctrl+Shift+X
Solution:
  1. Close VS Code
  2. Reopen it
  3. Wait 5 seconds
  4. Press Ctrl+Shift+X
  5. Check Installed section
```

### Extensions Not Working
```
Issue: Extensions installed but not functioning
Solution:
  1. Restart VS Code completely
  2. Open project folder
  3. Wait 10-15 seconds for extensions to activate
  4. Try using extension feature
  5. If still not working, reinstall specific extension
```

### Conflicts Between Extensions
```
Issue: Multiple formatters or linters conflicting
Solution:
  1. Set default formatter: Prettier
  2. Disable other formatters
  3. Restart VS Code
  4. Test again
```
